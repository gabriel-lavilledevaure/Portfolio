SAÉ 1.05 - Produire un site Web

Intégration & Développement Hébergement

Présentation du projet

Dans cette SAÉ, l’objectif était de concevoir, intégrer, développer et mettre en ligne un site One Page pour une start-up fictive. J’ai choisi de créer le site Diamond Wheels, une marque innovante dans le domaine de la mobilité urbaine. Ce choix de thématique m’a permis d’explorer un univers moderne et dynamique.

Le projet s’est appuyé sur un UI Kit Figma fourni, que j’ai adapté pour correspondre à l'identité visuelle de ma start-up. Chaque composant a été pensé pour favoriser une navigation fluide et intuitive. J’ai également fait le choix d’une approche mobile-first afin d’assurer une bonne accessibilité du site sur tous les supports, ce qui est aujourd’hui indispensable.

La structure du site repose sur une architecture modulaire : chaque section (héro, services, témoignages, contact, etc.) est un composant indépendant que j’ai intégré puis transformé en template réutilisable. Cette méthode de travail m’a permis de gagner en efficacité tout en facilitant la maintenance du site.

En phase de développement, j’ai séparé les données (JSON) du template HTML/CSS, et j’ai utilisé JavaScript pour automatiser l’affichage du contenu. Ce choix répond à une logique de durabilité et de performance : si demain les contenus changent, je peux les mettre à jour sans toucher au code source du site.

Enfin, j’ai mis en ligne mon site via une solution d’hébergement standard sur le serveur de PulseHeberge. Ce processus m’a permis de comprendre les étapes nécessaires à la publication d’un projet web complet.

Déroulement

  • Phase 1 : Intégration HTML/CSS des composants fournis dans le UI Kit sur Figma
  • Phase 2 : Génération dynamique du site à l’aide de templates HTML, JSON et JavaScript
  • Phase 3 : Mise en ligne sur un hébergement distant
